啰嗦几个点
1）创建含排序键的表示，指定key-schema时一定是先指定HASH这个类型再指定RANGE这个，不然一定报错
2）如需修改读写容量修改命令最后的provisioned-throughput下面对应值即可
3）就是需要提前设置连接服务器的环境变量如key及region，不然需命令行指定
